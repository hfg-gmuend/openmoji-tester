OpenMoji Tester App

Does your SVG file follow the OpenMoji Styleguide?



This is a small helper web app which lets you check quicky whether your OpenMoji svg files meet the technical requirements of the OpenMoji Styleguide. Every svg file is tested on things like: colors, layer, names, size etc.


1. Select svg-files

2. Check file structure

(According to styleguide specifications)

All of your files have to produce green tests only. Fix your files until everything is green! Please! :)

3. Check visually (for dark mode)
